### 待执行步骤（需手动完成）

#### 第一步：注册微信小程序（约10分钟）

1. 访问 https://mp.weixin.qq.com
2. 点击"立即注册" -> 选择"小程序"
3. 填写账号信息（邮箱、密码）
4. 邮箱激活
5. 信息登记（选择"企业"类型）
6. 主体信息填写（上传营业执照）
7. 管理员身份验证（微信扫码）
8. **费用**: 300元/年认证费

#### 第二步：下载开发者工具（约5分钟）

1. 访问 https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 选择对应系统版本下载
3. 安装并打开

#### 第三步：导入项目（约5分钟）

1. 打开微信开发者工具
2. 使用微信扫码登录
3. 点击"+"创建新项目
4. 项目目录：选择 `/workspace/super-assistant/miniprogram`
5. AppID：使用注册获得的AppID
6. 开发模式：小程序
7. 后端服务：不使用云服务（稍后配置）

#### 第四步：配置云开发（约10分钟）

1. 点击工具栏"云开发"按钮
2. 开通云开发（免费额度可用）
3. 创建环境名称：`super-assistant-env`
4. 配置数据库集合：
   - `chat_history` - 对话记录
   - `members` - 会员信息
   - `orders` - 订单记录
5. 上传云函数：
   - 右键 `cloudfunctions/ai-chat` -> 上传并部署
   - 右键 `cloudfunctions/payment` -> 上传并部署
   - 右键 `cloudfunctions/analytics` -> 上传并部署

#### 第五步：测试功能（约15分钟）

1. 编译小程序
2. 测试对话功能
3. 测试会员购买流程
4. 测试情绪分析
5. 修复发现的问题

#### 第六步：提交审核（约1-3天）

1. 点击"上传"按钮
2. 填写版本号：1.0.0
3. 填写项目备注
4. 登录小程序后台
5. 版本管理 -> 开发版本 -> 提交审核
6. 填写审核信息
7. 等待审核结果

#### 第七步：上线发布（约1小时）

1. 审核通过后
2. 点击"发布"
3. 填写发布说明
4. 正式上线
